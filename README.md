🛒 KHARIDO - AI-Powered Full Stack E-Commerce Store

📌 Overview

KHARIDO is a modern Full Stack & AI-Based E-Commerce Web Application built using React + Django.
It provides a seamless shopping experience with smart recommendations, fast UI, and scalable backend architecture.

---

🚀 Features

- 🔐 User Authentication (Login / Signup)
- 🛍️ Product Browsing & Search
- 🛒 Add to Cart & Order Management
- 💳 Secure Checkout System
- 📦 Order Tracking
- 🤖 AI-Based Product Recommendations
- 💡 Generative AI Features (Smart Suggestions / Chatbot Ready)
- ⚡ Fast & Responsive UI (Tailwind CSS)

---

🧰 Tech Stack

💻 Frontend

- React.js
- Tailwind CSS
- JavaScript

⚙️ Backend

- Django
- Django REST Framework

🗄️ Database

- PostgreSQL

🤖 AI Integration

- Generative AI APIs (Recommendations & Smart Features)

⚙️ Installation & Setup

1️⃣ Clone the Repository

git clone https://github.com/your-username/kharido.git
cd kharido

2️⃣ Backend Setup

cd backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver

3️⃣ Frontend Setup

cd frontend
npm install
npm run dev
